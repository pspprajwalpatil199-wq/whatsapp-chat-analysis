# WhatsApp Chat Analysis

A simple data science project to analyze WhatsApp chat data using Python and Streamlit.

## Features

* Upload WhatsApp chat file (.txt)
* View chat data in structured format
* Top statistics:

  * Total messages
  * Total words
  * Media shared
  * Links shared
* Most active users (bar chart + percentage table)
* Top 10 most common words (chart + table)

## Tech Stack

* Python
* Pandas
* Matplotlib
* Streamlit

##  How to Run

pip install -r requirements.txt
streamlit run main.py


##  Dataset

Export your WhatsApp chat as `.txt` and upload it in the app.

## Purpose

This project demonstrates data cleaning, analysis, and visualization using real-world chat data.
